# Markdown Web (MdWeb)

## 1. 概述

一个在线展示 Markdown 文件的极简方法。

## 2. 特点

- 简单易用。放在哪里，哪里的 Markdown 文件就会被渲染。
- Markdown 文件内嵌的超链接、图片等无需任何处理，都能正常工作。

## 3. 快速开始

- 放置 `.htaccess`、 `index.php` 到主机目录。
- 放置 `.md` 文件到主机目录，`default.md` 为默认的内容入口文件。
- 用浏览器访问 `.md` 文件，Markdown 已被渲染。

## 4. 其它

- 支持 KaTeX 公式。
- 支持 Mermaid 图表。
- 支持代码语法高亮。
